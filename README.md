## 📖 1. Descrição da Solução

O sistema proposto é uma plataforma de apoio à alfabetização de crianças e adolescentes internados no IMIP, utilizando IA generativa para criar conteúdos personalizados, respeitando o estado emocional, ritmo e contexto clínico do paciente.

A solução permite que professores hospitalares acompanhem o progresso pedagógico enquanto a criança interage com histórias e atividades adaptativas.
